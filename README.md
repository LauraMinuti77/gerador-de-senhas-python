# 🔐 Gerador de Senhas em Python

Este é um projeto simples desenvolvido em Python que gera senhas aleatórias seguras, utilizando letras, números e caracteres especiais.

## 💻 Tecnologias utilizadas
- Python

## ⚙️ Como funciona
O usuário informa o tamanho da senha desejada, e o programa gera automaticamente uma senha aleatória combinando:
- Letras maiúsculas e minúsculas
- Números
- Caracteres especiais

## ▶️ Como executar o projeto
1. Certifique-se de ter o Python instalado
2. Baixe o arquivo `gerador_senha.py`
3. Execute o código no terminal com o comando:

```bash
python gerador_senha.py
